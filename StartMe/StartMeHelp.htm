StartMe is a task management program that allows you to manage up to 9 tasks per configuration and a virtually unlimited number of configurations. You can:

        Set tasks to autostart

        Keep tasks running when the crash

        Sequential control

        Minimize on startup

        Administrative rights

        Priority

        Delay before startup

        Delay after startup

        CPU max usage before continuing,

        Send commands to windows after startup or before shutdown

        Stop in a required sequence (default to reversing startup sequence)

        Delay after stop

        Kill task if needed

        Multiple configurations for essentially unlimited task management

        Selection of a specific setting during startup (defaults to running all settings)

Running the first time

#1 Select programs using the file browser button on each Path box.

#2 Set Auto start if you want to be able to start tasks automatically when StartMe is started. You also will then have to check the Startup box at the bottom.

#3 You can set a sequence number to control the order of startup and the shutdown order (if you want shutdown to be anything other than the reverse of startup). If you dont put in a sequence number, then it will simply follow the task sequence 1-9 for startup and reverse it for shutdown.

#4 An easy way to prevent disk thrashing is to use the CPU % control to wait until the CPU dies down below some value (like 5% or 10% or whatever the idle state for a task shows for CPU usage).

#5 Delays can be programmed before start, after start, and after shutdown. Sometimes a program may need some extra time before starting another dependent program or may take a while to shut down, so you dont get a failed shutdown message. The start Delay is the maximum time before it will continue starting tasks. The stop Delay is the maximum time for shutdown before an error message is displayed.

Key entries

 

A limited set of key entries can be made to any window during startup or shutdown using this format:

WindowName KEY KEY KEY KEY..

WindowName must be the starting characters of any window enough to make it unique.

Some programs do not recognize standard shutdown commands and require keystrokes to shut down politely or must activate certain windows.

Some examples known to need this and working entries:

For Log4OM - "Awaiting confirmation" DELAY ENTER

Note: As of JTAlert 2.3.14 these keystrokes are no longer needed

"JTAlertX 2" ALT+A DELAY UP ENTER

Known key entries (if you need more keys please send request to mdblack98@yahoo.com).

        Cursor keys DOWN, UP, RIGHT, LEFT

        ENTER

        DELAY (delays 1 second)

        TAB

        ALT

        ALT+A

        ALT+F

        ALT+X

        ALT+F4

        Any keys entered not in the above list will be as-is

More than 9 tasks

You can have multiple Settings to allow many more than 9 tasks. Each setting will have its own page of 9 tasks. During startup all settings will be examined for autostart and the first setting name will be made the visible one after all tasks have been started. You can switch between settings from the Settings Name pulldown. If you pass a setting name as an argument to StartMe only that settings will be loaded, and other settings will be ignored.

To create a new settings page just type a name in the Settings Name box and hit enter.

Reordering Tasks

If you want to rearrange the tasks you can right-click in the Path and you will get the option to swap with one of the other tasks. All settings will be swapped between them.

Auto Start

At startup all configurations will be examined for autostart items by default. You can override this behavior by passing the configuration name (case sensitive) as an argument to StartMe.

Tasks

You can double-click on the number in front of the path to toggle the enabling of task. A task that is disabled will not be able to start at all.

By default tasks will be started from top to bottom on the selected configuration setting. You can change the start or stop order by enter the number 1-9 in the Start or Stop columns. The only time this should be needed is if you have to stop tasks in a different order the reverse of the startup. You can right-click on the task path to swap it with another task to change the default order.

If you enter a 0 or in the Stop column then that task will not be stopped when StopAll is clicked.

If you check the Auto box then all tasks in all configurations with Auto checked will start when StartMe starts. You can have StartMe select a setting to start by passing the setting name as an argument to StartMe. You can have multiple StartMes running if desired or only start one configuration and manually select the others.

Note: Shutdown of processes is not as robust as Id like. Shutdown works most of the time but there are some states that some programs get into where they wont shut down politely. For stubborn programs most can be killed with the Kill checkbox or Ctrl-click on the Stop button. Some programs cannot be killed politely as they will leave some processes running. So be careful with the Kill selection.

The Keep checkbox will restart a process if it crashes.

Configuration Backup & Moving to a new PC

Every time you make a change in your configuration and restart the program your user.config will get backed up. StartMe will tell you when your configuration has changed. Click the Backups button to see your backups and restore from them.

Backups will be located in %LOCALAPPDATA%\StartMe\StartMe.exe_StrongName..\1.0.0.0

If moving to a new PC copy the user.config file and run the program once to create the default user.config file and copy over the file created.
